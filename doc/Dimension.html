Dimension

new Dimension(name, dimension, dimension, dimension, multi)

mda/Dimension.js, line 14
A Dimension organizes a dataset along a particular property, or variation of a property. Defining a dimension with a function of:
function(d){ return d.Surname; }
will slice a dataset by the distinct values of the Surname property.
Name Type Description
name String The short name used to identify this dimension, and any linked dimensions sharing the same name
dimension function The function used to categorize points within the dimension.
