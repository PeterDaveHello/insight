group(dataset, name, groupFunction){object}

Dashboard.js, line 71
This method takes a dataset, name and grouping function, returning a Grouping with a Dimension created as part of that process.
Name Type Description
dataset object A crossfilter, for example, returned by the addData() method
name string A uniquely identifying name for the dimension along which this grouping is created. Used to identify identical dimensions in other datasets.
groupFunction function The function used to group the dimension along. Select the property of the underlying data that the dimension is to be defined on. Rounding or data manipulation can alter the granularity of the dimension
Returns:
- An aggregated Grouping that can be manipulated and have calculations applied to it
