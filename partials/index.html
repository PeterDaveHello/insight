Insight Charts

Create simple Javascript charts and analytical dashboards with insight.js

InsightJS is a data analysis and visualization toolset that lets you quickly explore multi-dimensional datasets. Using crossfilter and d3, insightJS allows aggregation of sums, averages and counts of particular values in your data set.

The visualizations are interactive, allowing the dataset to be sliced and diced along dimensions with all related charts updating to changes in data.

The example below uses data from the public iTunes API, allowing easy visualization of the patterns behind the data. Explore the charts by clicking on particular bars or bubbles to see if you can spot any interesting patterns in the App Store ecosystem.

Visualization

InsightJS provides many commonly used visualizations. Column, row, scatter, line and bubble series are currently available.

Analysis

Quickly and easily aggregate your data into more meaningful statistic to explore the patterns inside.
